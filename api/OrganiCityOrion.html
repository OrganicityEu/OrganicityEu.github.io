 query number (double)

Successful response

Body Schema: object

contextElements: object[]
Reference: ContextElement

Schema definitions

Attribute: object

The Attributes can be used to provide information about the available sensor measurements,

name: string , x ∈ { name , description , position , TimeInstant , .* }

Name of this attribute.

type: string , x ∈ { urn:oc:attributeType:name , urn:oc:attributeType:description , coords , urn:oc:datatype:ISO8601 , urn:oc:attributeType:.* }

Data type of this attribute.

value: string

The value of this attribute.

metadatas: object[]

A list of Metadata assosiated with this Attribute.

Reference: Metadata

ContextElement: object

id: string

The UUID of the ContextElement.

type: string , x ∈ { urn:oc:entityType:iotdevice , urn:oc:entityType:smartphone }

The type of the ContextElement.

isPattern: boolean , x ∈ { true , false }

Type of the ContextElement description. When isPattern is true in queries multiple ContextElements can be queried.

attributes: object[]

A list of Attributes assosiated with this ContextElement.

Reference: Attribute

Metadata: object

Metadata can be used to provide information about the units of measurements of the provided sensor measurements.

name: string , x ∈ { unit , datatype , location }

Name of the metadata

type: string , x ∈ { string , urn:oc:datatype:numeric }

Data type of the metadata.

value: string

value of metadata
